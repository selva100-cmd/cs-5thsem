*Software* 

To open VS Code:
Windows:
Click the Start menu (Windows icon).
Type Visual Studio Code or VS Code.
Click Visual Studio Code to open it.
Or:
Press Windows key → type VS Code → press Enter.
If you installed it but cannot find it:
Open File Explorer
Go to:

C:\Users\YourName\AppData\Local\Programs\Microsoft VS Code
Open Code.exe
After opening, you can create a folder for your HTML/PHP project and open it in VS Code.
Open VS Code
On the left side, click the Extensions icon (four small squares)
Shortcut: Ctrl + Shift + X
In the search box, type the extension name:
Live Server
PHP Intelephense
Prettier - Code formatter
Click the extension
Click Install
After installation, restart VS Code if needed
For your HTML + CSS + JavaScript + PHP setup, install: ✅ Live Server
✅ PHP Intelephense
✅ Prettier - Code formatter
✅ Auto Rename Tag
✅ HTML CSS Support
Visual Studio Code (recommended)
Open VS Code
Open your project folder
Right-click in Explorer → New File
Type:

index.html
Press Enter
Write your HTML code
To run your index.html file in VS Code:
Method 1: Using Live Server (HTML, CSS, JavaScript)
Open VS Code
Open your project folder
Click index.html
Right-click inside the file
Click Open with Live Server
Your browser will open and show the webpage
Method 2: Directly open
Go to your project folder
Double-click index.html
It opens in your browser
